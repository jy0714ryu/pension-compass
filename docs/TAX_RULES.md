## 1. 연금저축 (개인연금)

### 1.1 납입 시 세액공제

| 총급여 | 공제 한도 | 공제율 | 최대 공제액 |
|---|---|---|---|
| 5,500만원 이하 | 600만원 | 16.5% | 99만원 |
| 5,500만원 초과 | 600만원 | 13.2% | 79.2만원 |

> IRP 포함 시 한도 900만원 (연금저축 600 + IRP 300)

### 1.2 수령 시 과세 (연금소득세)

#### 분리과세 세율 (연간 1,500만원 이하)

| 수령 나이 | 세율 |
|---|---|
| 55~69세 | 5.5% |
| 70~79세 | 4.4% |
| 80세 이상 | 3.3% |

> 지방소득세 포함 세율

#### 종합과세 vs 분리과세 선택

- 연간 연금소득 1,500만원 초과 시 **선택 가능**
- 종합과세: 다른 소득과 합산하여 6~45% 누진세율 적용
- 분리과세: 위 세율로 원천징수 종결

**판단 기준:**
```
종합소득 과세표준 1,200만원 이하 → 종합과세 유리 (6%)
종합소득 과세표준 4,600만원 초과 → 분리과세 유리
```

### 1.3 중도 해지 시 과세 (기타소득세)

| 구분 | 세율 | 비고 |
|---|---|---|
| 세액공제 받은 원금 | 16.5% | 기타소득세 |
| 운용수익 | 16.5% | 기타소득세 |
| 세액공제 안 받은 원금 | 비과세 | |

## 2. IRP (개인형 퇴직연금)

### 2.1 납입 시 세액공제

- 연금저축과 합산 900만원 한도
- 공제율: 연금저축과 동일 (13.2% / 16.5%)

### 2.2 퇴직금 이전 시

- 퇴직금 IRP 이전: **퇴직소득세 이연**
- 연금 수령 시: 퇴직소득세 × 70% (30% 감면)

#### 퇴직소득세 계산 공식

```
퇴직소득 과세표준 = (퇴직급여 - 근속연수 공제) × 12 / 근속연수

[근속연수 공제]
- 5년 이하: 100만원 × 근속연수
- 10년 이하: 500만원 + 200만원 × (근속연수 - 5)
- 20년 이하: 1,500만원 + 250만원 × (근속연수 - 10)
- 20년 초과: 4,000만원 + 300만원 × (근속연수 - 20)
```

### 2.3 연금 vs 일시금 수령 비교

| 수령 방식 | 세금 | 특징 |
|---|---|---|
| 일시금 | 퇴직소득세 100% | 즉시 현금화 |
| 연금 (55세~) | 퇴직소득세 × 70% | 30% 감면, 분할 수령 |

## 3. ISA (개인종합자산관리계좌)

### 3.1 비과세 한도

| 가입자 유형 | 비과세 한도 |
|---|---|
| 일반형 | 200만원 |
| 서민형/농어민형 | 400만원 |

> 초과분: 9.9% 분리과세

### 3.2 의무 가입 기간

- **3년** (중도 해지 시 세제혜택 환수)

### 3.3 만기 후 연금저축 이전 (★핵심 절세 포인트)

| 조건 | 혜택 |
|---|---|
| 만기 후 60일 내 이전 | 이전액의 10% 세액공제 |
| 세액공제 한도 | 연 300만원 |

**계산 예시:**
```
ISA 만기액 3,000만원 → 연금저축 이전
세액공제: 3,000만원 × 10% = 300만원 (한도 도달)
실제 환급: 300만원 × 13.2% = 39.6만원 (고소득자 기준)
```

## 4. 인출 순서 최적화 원칙

### 4.1 일반 원칙

```
[최적 인출 순서 - 기본]
① ISA 만기분 (비과세 혜택 소진 후)
② 연금저축 (세액공제 안 받은 원금 먼저)
③ IRP (퇴직금 이전분)
④ 연금저축/IRP (세액공제 받은 원금 + 수익)
```

### 4.2 나이별 전략

| 나이 | 전략 |
|---|---|
| 55세 미만 | 해지 자제 (16.5% 기타소득세) |
| 55~59세 | 연 1,500만원 이하 유지 (분리과세) |
| 60~69세 | 5.5% 구간 최대 활용 |
| 70~79세 | 4.4% 구간 전환 |
| 80세 이상 | 3.3% 최저세율 |

### 4.3 시나리오별 최적화

#### 시나리오 A: 조기 퇴직자 (55세)
```
문제: 연금저축 3억, IRP(퇴직금) 2억, 월 생활비 500만원 필요
해결:
- 55~59세: ISA 만기분 + 연금저축 비공제분 우선 인출
- 60세~: IRP 연금 수령 개시 (30% 감면 적용)
- 연간 1,500만원 이하 유지 (분리과세)
```

#### 시나리오 B: ISA 만기 직전자
```
문제: ISA 4,000만원 만기 예정, 연금저축 1억 보유
해결:
① ISA 만기 후 60일 내 연금저축 이전 (세액공제 300만원 확보)
② 이전된 자금은 5년 후 연금 수령 개시
③ 기존 연금저축과 별도 관리 (선입선출 방지)
```

## 5. 주의사항 & 엣지 케이스

### 5.1 부분 해지 순서 (선입선출)

연금저축 해지 시 인출 순서:
```
① 세액공제 받은 납입금 (16.5% 과세)
② 운용수익 (16.5% 과세)
③ 세액공제 안 받은 납입금 (비과세)
```
> ⚠️ 세액공제 안 받은 원금을 먼저 빼고 싶어도 불가능

### 5.2 연금수령 한도

```
연금수령 한도 = 연금계좌 평가액 / (11 - 연금수령 연차) × 120%
```

- 1년차: 평가액 / 10 × 120%
- 10년차: 평가액 / 1 × 120% (전액 가능)

### 5.3 건강보험료 영향

| 소득 유형 | 건강보험료 |
|---|---|
| 분리과세 연금소득 | 피부양자 자격 유지 가능 |
| 종합과세 연금소득 | 지역가입자 전환 가능성 |

> 연간 2,000만원 이상 연금소득 시 건보료 부과 대상 (단, **공적연금**만 합산 —
> 사적연금은 제외. 상세 근거·요율·부과 로직은 [8. 건강보험료 (v1.1)](#8-건강보험료-v11) 참조)

## 6. 계산 공식 요약 (앱 구현용)

### 6.1 연금소득세 계산

```python
def calc_pension_tax(amount: int, age: int) -> int:
    """연금소득세 계산 (분리과세)"""
    if age >= 80:
        rate = 0.033
    elif age >= 70:
        rate = 0.044
    else:  # 55~69
        rate = 0.055
    
    return int(amount * rate)
```

### 6.2 ISA 이전 세액공제

```python
def calc_isa_transfer_credit(transfer_amount: int, income_level: str) -> int:
    """ISA 만기 후 연금저축 이전 세액공제"""
    credit_base = min(transfer_amount * 0.1, 3_000_000)  # 10%, 최대 300만원
    
    if income_level == "high":  # 총급여 5,500만원 초과
        rate = 0.132
    else:
        rate = 0.165
    
    return int(credit_base * rate)
```

### 6.3 퇴직소득세 (연금 수령 시)

```python
def calc_retirement_pension_tax(retirement_tax: int) -> int:
    """퇴직금을 연금으로 수령 시 세금 (30% 감면)"""
    return int(retirement_tax * 0.7)
```

## 7. 국민연금 (v1.1)

> 검증일: 2026-07-11 · v1.1 Task 1 (국민연금 크레바스 + 건보료)
> 근거: `docs/exec-plans/2026-07-11-v1.1-nps-crevasse.md`
> ⚠️ 본 섹션은 **간이 근사 산식** 용도다. 국민연금공단 실제 지급액은 가입기간 중 매년 재평가된
> 소득(재평가율 적용)·소득대체율 변경 경과조치(2025.12.31 이전/이후 가입기간 분리 계산) 등을
> 정밀 반영하므로 공단 공식 조회값과 차이가 날 수 있다. 앱은 항상 "간이 추정치" 임을 고지하고
> 공단 실측값 직접 입력 우회로를 제공한다 (exec-plan ① 참조).

### 7.1 기본연금액(노령연금) 산식

**표준 형태 확인됨:**

```
기본연금액 = 비례상수 × (A값 + B값) × [1 + 0.05 × (가입월수 - 240) / 12]   (가입기간 20년 초과 시)
기본연금액 = 비례상수 × (A값 + B값) × [1 - 0.05 × (240 - 가입월수) / 12]   (가입기간 20년 미만 시, 20년 기준 5%/년 감액)
```

- **★ 산식 결과 = 연액(원/년), 월액 = ÷12.** 검증: B=A(평균소득자)·40년 가입 시
  연액 = 1.29×2A×2 = 5.16A → 월액 = 5.16A/12 = 0.43A, 소득대체율 43% 정의와 정확히 일치.
  (연액을 월액으로 오독하면 12배 과대 추정 — 2026-07-11 Task 2 구현 중 실제 발생한 버그.)
- **A값**: 연금 수급 개시 직전 3년간 전체 가입자의 평균소득월액 평균 (매년 보건복지부 고시)
- **B값**: 가입자 본인의 가입기간 중 평균소득월액 (재평가율 적용, 앱 간이 계산에서는 "입력 월소득"으로 근사)
- **가입기간 20년 기준**: 20년 만근 시 계수 1.0 (100%), 이후 1년(12개월)마다 +5%, 20년 미만은 1년마다 -5%

### 7.2 비례상수·소득대체율 (★ 2026년 연금개혁 반영 — 확인 필요 항목)

| 항목 | 값 | 비고 |
|---|---|---|
| 소득대체율(40년 가입 기준), 2025.12.31까지 가입기간 | 41.5% | 종전 규정 (2028년까지 매년 0.5%p 인하 예정이었으나 개혁으로 중단) |
| 소득대체율(40년 가입 기준), **2026.1.1 이후 가입기간** | **43%** | 2025년 3월 국회 통과 「국민연금법」 개정 (2026-01-01 시행) |
| 비례상수 (소득대체율 40% 환산 기준) | 1.2 | NPS 공식 상수 (40% 소득대체율 시절 확정값) |
| 비례상수 (소득대체율 43% 환산, **파생값·미공식확인**) | **≈1.290** | `1.2 × (43/40)` 비례 산출. NPS가 43% 전용 신규 상수를 별도 고시했는지는 **미확인** — 공단은 2025.12.31 이전/이후 가입기간을 **분리 계산 후 합산**하는 방식(경과조치)을 쓸 가능성이 높음. |

**⚠️ 확인 필요 (앱 구현 시 보수적 처리 권장):**
공식 발표는 "소득대체율 43%"만 명시하고, 41.5%→43% 전환에 대응하는 새 비례상수를 NPS가
별도 고시한 자료는 이번 검색에서 찾지 못했다. 실제 공단 계산은 가입기간을 **2025-12-31 이전
(41.5% 상당 상수)과 2026-01-01 이후(43% 상당 상수)로 분리**해 각각 계산 후 합산하는 것이 원칙에
부합한다. v1.1 간이 계산기는 이 분리 계산을 구현하지 않고(YAGNI, 스코프 외) **단일 상수 1.290
(43% 환산치)으로 전체 가입기간을 근사**한다 — 이 경우 신규 가입자(전 가입기간이 2026년 이후)는
근사 오차가 작고, 장기 가입자(2025년 이전 가입기간 비중이 큼)는 **실제보다 소폭 과대 추정**될 수
있다. 결과 화면에 "간이 추정" 고지 필수.

## 8. 건강보험료 (v1.1)

> 검증일: 2026-07-11 · v1.1 Task 1. 지역가입자(은퇴 후 전환 가정) **소득 기준 부과분만** 다룬다.
> 재산·자동차 기준은 스코프 외(P1, 재산과표 선택 입력 보류)이며, 결과 화면에는 상시
> "재산(부동산 등) 기준분 제외한 소득 기준 추정치" 디스클레이머를 표기한다 (exec-plan②).

### 8.1 [★ 킬러 팩트 확정] 사적연금(연금저축·IRP)의 건보료 부과 여부

**결론: 현재(2026년) 사적연금(연금저축·IRP 개인 납입분)은 지역가입자 건강보험료 부과 대상
소득에 포함되지 않는다.** 건보료가 부과되는 것은 국민연금·공무원연금·사학연금·군인연금·
별정우체국연금 등 **5대 공적연금소득뿐**이다. → "연금계좌(연금저축·IRP) 인출은 건보료에
안 잡힌다"는 앱의 핵심 메시지는 **현재 실무 기준으로 성립한다.**

**단, 법적 불확실성 존재 (앱 문구에 신중한 단서 필요):**
- 건강보험법·소득세법 문언상으로는 사적연금소득도 부과 대상에 포함될 여지가 있다는 지적이
  있다 (감사원 2022년 지적, 국회 법제실도 위법 소지 언급 — "법과 현실의 괴리"). 현재는
  **행정 실무상 미부과**로 운영되는 상태.
- 국회에 사적연금 부과를 명시적으로 **면제**하는 방향의 개정안(김미애 의원 발의,
  「국민건강보험법 일부개정법률안」, 2025-08 발의)이 계류 중 — 이는 오히려 "현행 미부과 관행에
  법적 근거를 부여하려는" 방향이라 현재 결론을 강화하는 신호로 해석된다. 반대로 정부(복지부)는
  "장기적으로 부과 기반 확대는 검토 과제"라는 원칙적 입장도 병행 표명 — 즉 **중장기 부과 확대
  가능성 자체를 배제하지 않았다.**
- 퇴직연금(IRP 중 퇴직금 이전분)에 대해서도 별도 부과 확대 논란이 존재하나, 위 발의 법안은
  IRP·연금저축을 구분하지 않고 "사적연금" 전체를 대상으로 다룬다.

**앱 표기 권장 문구**: "현재(2026년) 기준 연금저축·IRP 인출은 건강보험료 부과 소득에 포함되지
않습니다. (제도 변경 가능성 있음)" — 단정적 "영구히 안전" 표현은 피할 것.

### 8.2 지역가입자 소득 보험료 산정 방식

| 항목 | 내용 |
|---|---|
| 부과 방식 | **정률제** — 소득 × 보험료율 (2022-09-01 2단계 개편으로 등급별 점수제에서 전환) |
| 공적연금소득 반영률 | **50%** — 공적연금소득 × 50% 만 건보료 부과 대상 소득에 산입 (예: 국민연금 연 1,200만원 → 부과 대상 600만원) |
