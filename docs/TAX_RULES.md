## 1. 연금저축 (개인연금)

### 1.1 납입 시 세액공제

| 총급여 | 공제 한도 | 공제율 | 최대 공제액 |
|---|---|---|---|
| 5,500만원 이하 | 600만원 | 16.5% | 99만원 |
| 5,500만원 초과 | 600만원 | 13.2% | 79.2만원 |

> IRP 포함 시 한도 900만원 (연금저축 600 + IRP 300)

### 1.2 수령 시 과세 (연금소득세)

#### 분리과세 세율 (연간 1,500만원 이하)

| 수령 나이 | 세율 |
|---|---|
| 55~69세 | 5.5% |
| 70~79세 | 4.4% |
| 80세 이상 | 3.3% |

> 지방소득세 포함 세율

#### 종합과세 vs 분리과세 선택

- 연간 연금소득 1,500만원 초과 시 **선택 가능**
- 종합과세: 다른 소득과 합산하여 6~45% 누진세율 적용
- 분리과세: 위 세율로 원천징수 종결

**판단 기준:**
```
종합소득 과세표준 1,200만원 이하 → 종합과세 유리 (6%)
종합소득 과세표준 4,600만원 초과 → 분리과세 유리
```

### 1.3 중도 해지 시 과세 (기타소득세)

| 구분 | 세율 | 비고 |
|---|---|---|
| 세액공제 받은 원금 | 16.5% | 기타소득세 |
| 운용수익 | 16.5% | 기타소득세 |
| 세액공제 안 받은 원금 | 비과세 | |

## 2. IRP (개인형 퇴직연금)

### 2.1 납입 시 세액공제

- 연금저축과 합산 900만원 한도
- 공제율: 연금저축과 동일 (13.2% / 16.5%)

### 2.2 퇴직금 이전 시

- 퇴직금 IRP 이전: **퇴직소득세 이연**
- 연금 수령 시: 퇴직소득세 × 70% (30% 감면)

#### 퇴직소득세 계산 공식

```
퇴직소득 과세표준 = (퇴직급여 - 근속연수 공제) × 12 / 근속연수

[근속연수 공제]
- 5년 이하: 100만원 × 근속연수
- 10년 이하: 500만원 + 200만원 × (근속연수 - 5)
- 20년 이하: 1,500만원 + 250만원 × (근속연수 - 10)
- 20년 초과: 4,000만원 + 300만원 × (근속연수 - 20)
```

### 2.3 연금 vs 일시금 수령 비교

| 수령 방식 | 세금 | 특징 |
|---|---|---|
| 일시금 | 퇴직소득세 100% | 즉시 현금화 |
| 연금 (55세~) | 퇴직소득세 × 70% | 30% 감면, 분할 수령 |

## 3. ISA (개인종합자산관리계좌)

### 3.1 비과세 한도

| 가입자 유형 | 비과세 한도 |
|---|---|
| 일반형 | 200만원 |
| 서민형/농어민형 | 400만원 |

> 초과분: 9.9% 분리과세

### 3.2 의무 가입 기간

- **3년** (중도 해지 시 세제혜택 환수)

### 3.3 만기 후 연금저축 이전 (★핵심 절세 포인트)

| 조건 | 혜택 |
|---|---|
| 만기 후 60일 내 이전 | 이전액의 10% 세액공제 |
| 세액공제 한도 | 연 300만원 |

**계산 예시:**
```
ISA 만기액 3,000만원 → 연금저축 이전
세액공제: 3,000만원 × 10% = 300만원 (한도 도달)
실제 환급: 300만원 × 13.2% = 39.6만원 (고소득자 기준)
```

## 4. 인출 순서 최적화 원칙

### 4.1 일반 원칙

```
[최적 인출 순서 - 기본]
① ISA 만기분 (비과세 혜택 소진 후)
② 연금저축 (세액공제 안 받은 원금 먼저)
③ IRP (퇴직금 이전분)
④ 연금저축/IRP (세액공제 받은 원금 + 수익)
```

### 4.2 나이별 전략

| 나이 | 전략 |
|---|---|
| 55세 미만 | 해지 자제 (16.5% 기타소득세) |
| 55~59세 | 연 1,500만원 이하 유지 (분리과세) |
| 60~69세 | 5.5% 구간 최대 활용 |
| 70~79세 | 4.4% 구간 전환 |
| 80세 이상 | 3.3% 최저세율 |

### 4.3 시나리오별 최적화

#### 시나리오 A: 조기 퇴직자 (55세)
```
문제: 연금저축 3억, IRP(퇴직금) 2억, 월 생활비 500만원 필요
해결:
- 55~59세: ISA 만기분 + 연금저축 비공제분 우선 인출
- 60세~: IRP 연금 수령 개시 (30% 감면 적용)
- 연간 1,500만원 이하 유지 (분리과세)
```

#### 시나리오 B: ISA 만기 직전자
```
문제: ISA 4,000만원 만기 예정, 연금저축 1억 보유
해결:
① ISA 만기 후 60일 내 연금저축 이전 (세액공제 300만원 확보)
② 이전된 자금은 5년 후 연금 수령 개시
③ 기존 연금저축과 별도 관리 (선입선출 방지)
```

## 5. 주의사항 & 엣지 케이스

### 5.1 부분 해지 순서 (선입선출)

연금저축 해지 시 인출 순서:
```
① 세액공제 받은 납입금 (16.5% 과세)
② 운용수익 (16.5% 과세)
③ 세액공제 안 받은 납입금 (비과세)
```
> ⚠️ 세액공제 안 받은 원금을 먼저 빼고 싶어도 불가능

### 5.2 연금수령 한도

```
연금수령 한도 = 연금계좌 평가액 / (11 - 연금수령 연차) × 120%
```

- 1년차: 평가액 / 10 × 120%
- 10년차: 평가액 / 1 × 120% (전액 가능)

### 5.3 건강보험료 영향

| 소득 유형 | 건강보험료 |
|---|---|
| 분리과세 연금소득 | 피부양자 자격 유지 가능 |
| 종합과세 연금소득 | 지역가입자 전환 가능성 |

> 연간 2,000만원 이상 연금소득 시 건보료 부과 대상

## 6. 계산 공식 요약 (앱 구현용)

### 6.1 연금소득세 계산

```python
def calc_pension_tax(amount: int, age: int) -> int:
    """연금소득세 계산 (분리과세)"""
    if age >= 80:
        rate = 0.033
    elif age >= 70:
        rate = 0.044
    else:  # 55~69
        rate = 0.055
    
    return int(amount * rate)
```

### 6.2 ISA 이전 세액공제

```python
def calc_isa_transfer_credit(transfer_amount: int, income_level: str) -> int:
    """ISA 만기 후 연금저축 이전 세액공제"""
    credit_base = min(transfer_amount * 0.1, 3_000_000)  # 10%, 최대 300만원
    
    if income_level == "high":  # 총급여 5,500만원 초과
        rate = 0.132
    else:
        rate = 0.165
    
    return int(credit_base * rate)
```

### 6.3 퇴직소득세 (연금 수령 시)

```python
def calc_retirement_pension_tax(retirement_tax: int) -> int:
    """퇴직금을 연금으로 수령 시 세금 (30% 감면)"""
    return int(retirement_tax * 0.7)
```
