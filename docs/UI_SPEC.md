# UI/UX 명세서

> 연금나침반 화면 설계

---

## 1. 화면 구성

```
[앱 구조]
├── 홈 (입력 폼)
├── 결과 (시뮬레이션)
├── 절세 팁 (카드 리스트)
└── 설정
```

---

## 2. 홈 화면 (입력 폼)

### 2.1 레이아웃

```
┌─────────────────────────────────────┐
│  🧭 연금나침반                      │
│  "최적의 인출 순서를 찾아드립니다"    │
├─────────────────────────────────────┤
│                                     │
│  📊 내 연금 자산                     │
│  ┌─────────────────────────────┐   │
│  │ 연금저축 잔액    [________]만원│   │
│  │ └ 세액공제 받은 금액 [____]만원│   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ IRP 잔액        [________]만원│   │
│  │ └ 퇴직금 이전분   [____]만원  │   │
│  └─────────────────────────────┘   │
│  ┌─────────────────────────────┐   │
│  │ ISA 만기 예정액  [________]만원│   │
│  └─────────────────────────────┘   │
│                                     │
│  👤 기본 정보                        │
│  ┌─────────────────────────────┐   │
│  │ 현재 나이          [__]세    │   │
│  │ 연간 목표 인출액 [______]만원 │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │     🔍 최적화 시뮬레이션      │   │
│  └─────────────────────────────┘   │
│                                     │
│         [광고 배너 영역]             │
└─────────────────────────────────────┘
```

### 2.2 입력 필드 상세

| 필드 | 타입 | 필수 | 기본값 | 유효성 |
|---|---|---|---|---|
| 연금저축 잔액 | 숫자 | ✅ | 0 | 0 이상 |
| 세액공제 받은 금액 | 숫자 | ❌ | 잔액과 동일 | ≤ 연금저축 잔액 |
| IRP 잔액 | 숫자 | ❌ | 0 | 0 이상 |
| 퇴직금 이전분 | 숫자 | ❌ | IRP 잔액과 동일 | ≤ IRP 잔액 |
| ISA 만기 예정액 | 숫자 | ❌ | 0 | 0 이상 |
| 현재 나이 | 숫자 | ✅ | 55 | 20~100 |
| 연간 목표 인출액 | 숫자 | ✅ | 2400 | > 0 |

### 2.3 UX 힌트

- 세액공제 금액 입력 시 "대부분 전액 공제 받으셨으면 그대로 두세요" 툴팁
- 숫자 입력 시 천 단위 콤마 자동 포맷
- 나이 입력 시 "55세 미만은 중도 해지 기타소득세 16.5% 적용" 경고

### 3.2 차트 스펙

| 차트 | 라이브러리 | X축 | Y축 | 시리즈 |
|---|---|---|---|---|
| 누적 세금 비교 | fl_chart LineChart | 연차 | 누적 세금(만원) | 최적/기존 |
| 인출 구성 | fl_chart PieChart | - | - | 계좌별 비율 |

### 3.3 애니메이션

- 차트 진입 시 라인 그려지는 애니메이션 (1.5초)
- 절감액 카운트업 애니메이션 (0 → 680만원)
- 순서 카드 페이드인 (순차적으로)

## 6. 반응형 고려

| 화면 | 폭 | 레이아웃 |
|---|---|---|
| 소형 (SE) | < 375 | 1열, 패딩 축소 |
| 표준 | 375~414 | 1열, 기본 패딩 |
| 대형 (Pro Max) | > 414 | 1열, 여유 패딩 |
| 태블릿 | > 600 | 2열 그리드 가능 |

---

## 7. 접근성

- 텍스트 최소 크기: 14sp
- 색상 대비: WCAG AA 이상
- 터치 영역: 최소 48x48dp
- 스크린 리더: 차트에 대체 텍스트 제공

---

## 8. 광고 배치 전략

| 위치 | 타입 | 노출 시점 |
|---|---|---|
| 홈 하단 | 배너 (320x50) | 상시 |
| 결과 진입 | 전면 | 시뮬레이션 완료 후 (5초 딜레이) |
| 절세 팁 하단 | 네이티브 | 스크롤 시 |

**광고 정책:**
- 결과 확인 전 광고 노출 금지 (UX 저해)
- 전면 광고는 세션당 1회로 제한
- 프리미엄 구독 시 광고 제거
