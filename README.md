# 연금나침반 - ISA 연금저축 세금 계산기 & 인출 시뮬레이터

> **PensionCompass**  
> 연금저축/IRP/ISA 인출 순서 최적화 시뮬레이터

## 🎯 핵심 가치 제안

**"같은 금액을 인출해도, 순서만 바꾸면 세금이 수백만 원 달라집니다"**

기존 연금 계산기들이 "얼마 인출하면 세금이 얼마"만 알려줄 때,  
연금나침반은 **"어떤 순서로 인출해야 세금이 최소화되는지"**를 시뮬레이션합니다.

---

## 📊 MVP 핵심 기능 (v1.0)

### 1. 인출 순서 최적화 시뮬레이터
- 입력: 연금저축 잔액, IRP 잔액, ISA 만기 예정액, 나이, 연간 목표 인출액
- 출력: 최적 인출 순서 + 예상 절감 세금액 + 연도별 시뮬레이션 그래프

### 2. 세금 계산기 (기본)
- 연금소득세 계산 (3.3%~5.5% 분리과세 vs 종합과세)
- ISA 만기 후 연금저축 이전 시 세액공제 계산
- 퇴직소득세 vs 연금소득세 비교

### 3. 절세 팁 카드
- 상황별 절세 전략 카드 UI (예: "55세 이전 해지 시 기타소득세 16.5%")

## 🛠 기술 스택

| 영역 | 기술 | 이유 |
|---|---|---|
| **프레임워크** | Flutter | iOS/Android 동시 배포, 빠른 개발 |
| **상태관리** | Riverpod | 심플하고 테스트 용이 |
| **로컬 저장** | Hive / SharedPreferences | 서버 없이 온디바이스 |
| **차트** | fl_chart | Flutter 네이티브, 커스텀 용이 |
| **광고** | Google AdMob | 배너 + 전면 광고 |

### 왜 Flutter?
- 1인 개발로 iOS/Android 동시 커버
- Hot Reload로 빠른 UI 이터레이션
- 대장님 기존 Python 백엔드 지식과 분리 (앱은 프론트만)

 ├─ IRP 잔액: [____만원]

## ⚖️ 핵심 세법 규정 (요약)

### 연금소득세
| 수령 나이 | 세율 (분리과세) |
|---|---|
| 55~69세 | 5.5% |
| 70~79세 | 4.4% |
| 80세 이상 | 3.3% |

> 연간 1,500만원 초과 시 종합과세 선택 가능

### ISA 만기 후 연금저축 이전
- 만기 후 60일 내 연금저축 이전 시 **이전액의 10% 세액공제** (최대 300만원)
- 비과세 혜택 + 추가 세액공제 = 더블 절세

### 퇴직소득 vs 연금소득
- 일시 수령: 퇴직소득세 (근속연수 공제 후 과세)
- 연금 수령: 퇴직소득세 × 70% (30% 감면)
